# Football Ticket Booking System

A database design and SQL assignment built around a simplified football ticket booking platform.

## What's Included

- **ERD** — Entity Relationship Diagram showing the relationships between Users, Matches, and Bookings
- **QUERY.sql** — SQL file containing table creation, sample data, and 7 queries covering joins, subqueries, null handling, pattern matching, and pagination

## Tables

- `users` — fans and ticket managers
- `matches` — football match details and ticket availability
- `bookings` — individual ticket purchases linking users to matches
